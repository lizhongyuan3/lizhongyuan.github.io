npm-publish

上传npm

  1. 注册npm账号 地址
  2. 在项目的目录下执行 npm login ==镜像源要切换回npm==
  3. npm publish 输入用户名 密码 邮箱即可 ==账号要是验证过的 npm 官网上先验证邮箱==
  4. npm unpublish xx –force 删除npm包 ==可能要动态验证码 需要手机下载Authenticator==
